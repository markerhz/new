# GemVerse

Genre

Modern Indie Match-3 Roguelite

Core Vision

GemVerse is not about matching gems.

Players live inside a tiny universe.

The spaceship is home.

The mining machine is the heart of the ship.

Crystals are discoveries, not candy.

Every mechanic, animation and visual decision should reinforce this feeling.

Core Pillars

- Cozy
- Living World
- Discovery
- Strategy
- Replayability

Planet Exploration Direction (Future)

- การสำรวจบนดาวเป็นฉาก Side-scrolling 2D Pixel Art ขนาดกระชับ
- ผู้เล่นเดินสำรวจ จุดโต้ตอบ และทางลับเพื่อเข้ามินิเกมหรือค้นหาไอเทมช่วยเหลือ
- รางวัลจากการสำรวจต้องกลับมาช่วยการวางแผนใน Match-3 อย่างชัดเจน
- ตัวเอกประจำโหมดสำรวจเป็นนักสำรวจหญิงสวมแว่น ผมดำดัดลอน และชุดอวกาศสีพาสเทล
- เริ่มจาก Vertical Slice หนึ่งฉาก หนึ่งมินิเกม และหนึ่งรางวัล ก่อนขยายเป็นหลายดาว

Never

- Feel like a casino
- Copy Candy Crush
- Use harsh neon
- Feel like a generic mobile game
